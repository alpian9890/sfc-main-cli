# SFC Main CLI

Public installer and binary release channel for SeoFast CLI.

Source code is kept private. This repository only provides the installer script and release binaries.

## Install

curl -fsSL https://raw.githubusercontent.com/alpian9890/sfc-main-cli/main/scripts/install.sh | sh

Alternative:

wget -qO- https://raw.githubusercontent.com/alpian9890/sfc-main-cli/main/scripts/install.sh | sh

## Update

seofast update

## Release Assets

- seofast-linux-x64
- seofast-linux-arm64
